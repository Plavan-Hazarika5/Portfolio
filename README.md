# Plavan Portfolio

A modern developer portfolio built with React, Vite, Tailwind CSS, and GSAP animations.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS
- **Animation**: GSAP, Framer Motion, Lenis
- **Core**: HTML, CSS, JavaScript
- **Backend (shown in stack)**: Node.js, Express, MongoDB
- **Tools**: Git, GitHub
- **Design (shown in stack)**: UI/UX, Graphic Design, Figma, Framer
- **Languages (shown in stack)**: Python, Java, C, C++
